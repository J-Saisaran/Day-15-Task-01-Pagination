# Day 15 Task 01: Pagination

This project demonstrates pagination functionality using JavaScript. It fetches data from an external JSON file and displays it in a table format with pagination controls to navigate through different pages.

## Features

- **Pagination:** Allows users to navigate through pages of data with buttons for each page.
- **Dynamic Table:** Displays data fetched from a JSON file in a table format.
- **Responsive Design:** Ensures the table and pagination controls are displayed correctly on various screen sizes.

## Project Structure

- `index.html`: The main HTML file containing the structure of the table and pagination controls.
- `styles.css`: CSS file for styling the table and pagination controls to ensure a clean layout and responsiveness.
- `script.js`: JavaScript file responsible for fetching data, handling pagination, and updating the table.
